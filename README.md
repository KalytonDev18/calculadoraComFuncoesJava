# calculadora com funcoes em Java

-Projeto desenvolvido para praticar lógica de programação, métodos, tratamento de exceções e organização de código em Java.

Funcionalidades:

-Soma
-Subtração
-Multiplicação
-Divisão
-Tratamento de entradas inválidas

Tecnologias utilizadas:

-Java
-Scanner
-Exception Handling

Objetivo:

-Este projeto faz parte da minha jornada de aprendizado em desenvolvimento backend com Java.

Durante este projeto pratiquei:

-Estruturação de métodos
-Reutilização de código
-Organização de lógica
-Tratamento de erros com try/catch
